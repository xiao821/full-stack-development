# 前端（Vue 3 + Vite）

使用 Axios + Element Plus 的最小化示例工程（可随时替换为 Ant Design Vue）。

## 使用说明

1) 可选：在 `.env` 中设置后端 API 地址（默认 `http://localhost:3000`）：

```
VITE_API_URL=http://localhost:3000
```

2) 安装依赖并启动开发服务器：

```
npm install
npm run dev
```

打开浏览器访问 `http://localhost:5173`。

页面示例包含：
- 物品列表展示（表格）
- 新增物品输入框与按钮
- 后端健康状态指示
